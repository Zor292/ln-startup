# 🚔 LiveZone Discord Bot

بوت ديسكورد خاص بسيرفر LiveZone — سيرفر رول بلاي روبلوكس

---

## 📋 متطلبات الإعداد

### المتغيرات في Railway (Variables):

| Variable | من أين تحصل عليه |
|----------|----------------|
| `DISCORD_TOKEN` | Discord Developer Portal → Bot → Token |
| `CLIENT_ID` | Developer Portal → General Information → Application ID |
| `GUILD_ID` | كليك يمين على السيرفر → Copy Server ID |
